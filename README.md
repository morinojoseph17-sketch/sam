# Pink Personal Website

This is a GitHub Pages-ready personal website.

## Files
- `index.html` - main website file
- `.nojekyll` - tells GitHub Pages to publish the site as plain static HTML

## How to publish on GitHub Pages
1. Create a new GitHub repository.
2. Upload these files to the repository root.
3. Go to Settings > Pages.
4. Under Build and deployment, choose:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: /root
5. Save.
6. Wait for GitHub Pages to generate your live website link.
